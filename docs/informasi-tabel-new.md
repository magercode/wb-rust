# WibuScript (Wb-Rust)

## Filosofi Bahasa
Konsepnya:
- Keyword absurd campuran antara indonesia gaul dan jepang gaul karena mengambarkan kewibuan 
- Syntax sederhana (mirip Python) supaya interpreter mudah dibuat di Rust. 
- Untuk versi Legacy sebelumnya wajib kurawal dan banyak error memory leak!!
- Ga banyak simbol rumit

## Kata kunci WibuScript (New Style)
| Keyword    | Arti                      | Asal        |
| ---------- | ------------------------- | ----------- |
| `bikin`    | deklarasi variabel        | gaul        |
| `butuh`    | impor modul               | gaul        |
| `ekspor`   | ekspor modul              | gaul        |
| `fun`      | membuat fungsi            | jepang      |
| `kalo`     | if                        | gaul        |
| `ato`      | else                      | gaul        |
| `bentar`   | while loop                | gaul        |
| `balikin`  | return                    | gaul        |
| `baka`     | print/output              | jepang      |
| `true`     | boolean true              | programming |
| `false`    | boolean false             | programming |
| `nil`      | nilai kosong              | programming |
| `lanjut`   | continue                  | indo        |
| `berhenti` | break                     | indo        |


## Built-In (Rencana)
| fungsi      | fungsi              |
| ----------- | ------------------- |
| `baka()`    | print               |
| `bakaf()`   | print format        |
| `format()`  | format string       |
| `input()`   | input user          |
| `panjang()` | length              |
| `tipe()`    | type                |
| `angka()`   | cast ke number      |
| `teks()`    | cast ke string      |

## Error catch
| keyword  | arti    |
| -------- | ------- |
| `nani`   | error   |
| `yamete` | stop    |
| `sugoi`  | success |

<summary>Tentang Wibu Interpreter</summary>

> WibuCPP adalah bahasa pemrograman yang terinspirasi dari JavaScript, Python, dan Bash, dengan sintaks yang absurd dan mudah dipahami. Bahasa ini dirancang untuk sekedar bersenang-senang dan bereksperimen dengan konsep bahasa pemrograman.

</details>

<details>

<summary>Fitur</summary>

- Sintaks yang absurd dan gak tau
- males ngetik

</details>

<details>
<summary>Contoh Kode</summary>

## DASAR 
```wibu
// Ini komentar
bikin nama = "Wibu"
bikin umur = 17
baka "Nama: " + nama
baka "Umur: " + umur
```

## CONTOH KEDUA
```wibu

// function
moshi hai(namalu)
{
    baka "Konnichiwa, " + namalu + "!"
}

```

## CONTOH KETIGA
```wibu

bikin online = true
bikin nama = "Wibu"
bikin umur = 23

kalo online {
    baka "online!"
    baka "Nama: " + nama
    baka "Umur: " + umur
}

//loop
bikin n = 0
bentar (n < 5)
{
    baka n
    n = n + 1
}
```
